13. EDITOR BEHAVIOR

Write screen rules:

- AI output streaming
- user edit always editable
- no locked output
- auto-save every change
- debounce 1–2 sec local save
- autosync 30–60 sec

14. RESPONSIVE BREAKPOINTS

- bottom nav
- single column

Tablet

- optional side panel (chat history)

Desktop

Chat | Editor | Brain

15. PERFORMANCE RULES

- no heavy animation
- no shadow excessive
- no blur glass effects
- minimal DOM updates
- vanilla JS only
- lazy render chat history

16. UX FLOW UTAMA

Writing Flow

Prompt → AI → Edit → Train → Save → Sync

Chat Flow

Message → Stream → Save → Summarize

Learning Flow

Edit / Chat / Raw → Extract → Brain Update

- 1–2 tap untuk semua aksi utama
