18. Update v1.1 (UI/UX Enhancement)

Komponen UI tambahan yang diaplikasikan:
- **Touch Target**: Tombol navigasi dan menu diperbesar (_min-height 48px_) untuk aksesibilitas *mobile*.
- **Toasts**: Notifikasi sekarang dibedakan warnanya (Success = Hijau, Error = Merah) dengan animasi melayang dari bawah.
- **Copy Button**: Tombol "📋 Copy" disematkan pada sudut setiap _chat bubble_ AI.
- **Word Count**: Penghitungan karakter dan kata otomatis ditampilkan pada panel _editor footer_.
- **Persist State**: Menggunakan _localStorage_ untuk kembali ke tab terakhir saat dimuat ulang (Refresh).
- **Undo System**: Shortcuts `Ctrl+Z` & `Ctrl+Y` diimplementasikan secara kustom di layar Write agar interaksi input manual pengguna dan perubahan dari AI dapat dikembalikan dengan aman.
- **Think Filter**: Semua blok `<think>...</think>` dalam respons AI disembunyikan secara *real-time* saat perenderan Markdown.

---

19. Update v1.2 (Topbar & Modals)

Perubahan visual diterapkan pada elemen struktural:
- **Topbar**: Elemen UI `Workspace` berubah wujud dari tombol blok kiri, menjadi pill button (compact) di kluster kanan sejajar dengan indikator status. Pojok kiri kini eksklusif untuk judul aplikasi `GhostWriter`.
- **Modals**: Custom modal (`#confirm-modal` dan `#prompt-modal`) diperkenalkan menggantikan UI blocking *browser native*. Memiliki desain kartu transparan tengah dengan tombol (Batal / Ya).
- **Icon Update**: Ikon di tab Chat diganti dari `☷` ke ikon berbentuk jam `🕒`.
- **Outline WCAG**: Focus outline biru standar di-*override* menjadi outline _accent color_ solid tebal untuk panduan visual keyboard (*accessibility*).

---

20. Update v1.3 (Advanced UI Refactor)

Perombakan UI mendalam:
- **Brain CRUD UI**: Tab Style, Thinking, dan Memory kini dilengkapi tombol "Edit" dan "Hapus" pada setiap itemnya (`.insight-meta`).
- **Proposal Bulk Actions**: Panel khusus di atas daftar Proposal yang memunculkan dua *button* "Setujui Semua" dan "Tolak Semua" secara responsif (ditampilkan `display:flex` hanya saat ada data proposal yang masuk).
- **Sync Dialog (#sync-modal)**: Saat menekan tombol Sinkronisasi GitHub, sistem menahan layar dan menampilkan dialog *glassmorphism* dengan 2 tombol utama: `Kirim ke GitHub (Push)` dan `Ambil dari GitHub (Pull)`, guna menghindari sinkronisasi yang berbenturan secara tidak sadar.
- **AI Model Status (#model-status)**: Info AI tidak lagi menampilkan *hardcoded fallback* (mis. Qwen3-4B), melainkan langsung menyajikan model aktual (OpenRouter, Groq, dll) yang terekstrak dari config API key secara *realtime*.
