GhostWriter HF Space Architecture v1.0

1. Visi Produk

GhostWriter adalah aplikasi penulisan berbasis AI yang berjalan di Hugging Face Space, yang berfungsi sebagai sistem pembelajaran gaya menulis dan pola berpikir pengguna.

Tujuan utama:

- Mempelajari gaya menulis pengguna
- Mempelajari cara berpikir pengguna
- Meningkatkan kualitas tulisan berdasarkan data pengguna
- Menggabungkan AI + personal writing system

Bukan sekadar chatbot, tetapi sistem “Writing Intelligence Personal”.

---

2. Prinsip Desain

Sistem ini dibangun dengan prinsip:

- Cepat
- Ringan
- Mobile-first
- Single user
- Workspace-based
- Offline-friendly (partial)
- GitHub-backed storage
- Vanilla frontend (no framework heavy)

20. Update v1.1 (UI/UX & Feature Enhancement)

Peningkatan yang ditambahkan:
- **UI/UX**: Perbaikan touch target untuk mobile (min-height 48px), perbaikan padding/margin, animasi toast error/success.
- **State Persistence**: Menyimpan status tab aktif (Write, Chat, Brain) di `localStorage`.
- **Custom Undo/Redo**: Sistem Undo/Redo kustom (Ctrl+Z/Ctrl+Y) untuk membedakan perubahan AI dan input user tanpa merusak browser native undo.
- **Limitless Code**: Menghapus `max_length` constraint pada Pydantic backend, memungkinkan user paste kode atau prompt tanpa batas (tergantung context window AI).
- **AI Think Filter**: Membuang block `<think>...</think>` (contoh dari DeepSeek) agar UI tetap bersih.
- **Word Count & Copy Action**: Penambahan penghitung karakter/kata di editor, dan tombol salin cepat di chat.

---

21. Update v1.2 (Topbar & Data Management)
- **Topbar Refactor**: Pemindahan *workspace-button* ke *status-cluster* sebagai bentuk simplifikasi navigasi, dan penambahan teks _branding_ 'GhostWriter' di kiri atas.
- **Data Export/Import Offline**: Migrasi konsep *Snapshot* murni menjadi fitur Import/Export ZIP offline via `/api/import` dan `/api/export`. Konflik nama saat import ditangani dengan pembuatan workspace duplikat.
- **Custom Confirm Dialog**: Penggantian `window.confirm` dan `window.prompt` native dengan modal custom (glassmorphism) `.dialog-card` yang mendukung `Promise` di frontend.
- **Aksesibilitas (WCAG)**: Implementasi pseudo-class `:focus-visible` untuk _keyboard navigation_.
